Same.Energy Android Client

An elegant, high-performance Flutter Android client for same.energy — the AI-powered visual search engine. Built with Glassmorphism design and Riverpod state management.

Research Project

HERMES

A hybrid AI/ML network security system designed and deployed on ARM clusters for resource-constrained edge environments.

Challenge

Existing intrusion detection systems rely on GPU-backed deep learning or signature-only approaches — both impractical for edge environments with strict power, memory, and compute budgets.

What Was Built

A two-stage hybrid detection pipeline combining signature filtering with a quantized 3-layer deep neural network. The system runs on ARM cluster nodes, enabling distributed monitoring and cooperative intelligence sharing across the edge.

94.7% Novel attack detection
4.2W Power per node
15 kpps Throughput
89.3% Adversarial robustness

Research

Published work.

HERMES: Design and Deployment of a Hybrid AI/ML Network Security System on ARM Clusters for Edge Environments

Arpankumar G. Raval & Devgna Vyas

7th International Conference on Soft Computing and Its Engineering Applications (icSoftComp 2025)

📅 December 9–11, 2025 📚 Published by Springer Nature 🏛️ CHARUSAT University

A hybrid intrusion detection system combining signature-based filtering with a quantized deep neural network, deployed on ARM-based edge clusters. Achieves 94.7% detection of novel attacks at just 4.2W per node with 89.3% adversarial robustness — 67% less energy than comparable x86 setups.
